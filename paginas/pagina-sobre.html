Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de apresentar informações sobre dinossauros de forma simples, visual e educativa. Aqui é possível explorar diferentes grupos, como carnívoros, herbívoros e onívoros, conhecendo características e curiosidades sobre essas criaturas pré-históricas.

Além do conteúdo educativo, o projeto também faz parte dos meus estudos em desenvolvimento web, utilizando tecnologias como HTML, CSS e JavaScript para construção da interface e organização das páginas.

O objetivo é praticar conceitos de desenvolvimento front-end, responsividade, organização de código e criação de experiências interativas para o usuário.
